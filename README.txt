# TestNanoDev
create a venv with python3.8 and activate him
git clone this repo
place yourself at the same level as manage.py
install all packages with requirements.txt file
configure the mongodb database by creating a new one with the name testnanodev
run all the migrations by the command "python manage.py migrate"
then start the server with "python manage.py runserver"
the main endpoint is "localhost:8000/News"
to get a specify news, you can just add the id like "localhost:8000/News/3"
